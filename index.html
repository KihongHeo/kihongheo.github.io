Research Interest

My research aims to develop program reasoning systems for safe and reliable software. In particular, I am working on the following topics:
  • Sound, Scalable, Precise & User-friendly Static Program Analysis
  • Effective Machine Learning Techniques for Program Analysis
  • Synergistic Approaches to Software Engineering by Logical and Probabilistic Reasoning
  • Semantic-based Software Security
