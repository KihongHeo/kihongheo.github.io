Research Interest

My research aims to develop program reasoning systems for safe and reliable software. In particular, I am working on the following topics:
  • AI-based program analysis system for detecting deep semantic software bugs
  • General-purpose program debloating system for secure and efficient software
  • Scalable program synthesis system for automatic software generation and repair
For more information about my research, see projects, publications, and talks.

Software

I have contributed to the following open-source software:
  • Chisel: an automated program debloating system
  • Sparrow: a static analyzer for C program
  • Petablox: declarative program analysis framework for Big Code
  • Infer: a static analyzer for Java, C, C++, and Objective-C
  • Euphony: a probabilistic model-guided program synthesizer
